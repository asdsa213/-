# 班课作业通微信小程序

这是一个微信小程序原生开发项目，用于班级作业通知、课程查看、学生绑定、管理员发布作业和课代表协作。

## 技术栈

- 微信小程序原生开发
- WXML / WXSS / JavaScript / JSON
- 微信云开发
- 无第三方依赖

## 主要功能

- 学生绑定姓名、学号、班级
- 支持一个学生加入多个班级并切换查看
- 首页按班级显示未截止作业
- 课程页按班级显示课程
- 课程历史页查看历史作业
- 管理员发布、修改、删除作业
- 管理员管理班级、课程、课代表
- 课代表可发布/修改对应课程作业
- 作业截止状态自动计算

## 使用说明

1. 使用微信开发者工具导入本项目。
2. 开通微信云开发。
3. 创建并配置对应数据库集合和云函数。
4. 上传部署 `cloudfunctions` 目录下的云函数。
5. 在 `app.js` 中确认云环境 ID 正确。

## 注意

`project.private.config.json` 是本地开发者工具私有配置，不建议提交到 GitHub。
